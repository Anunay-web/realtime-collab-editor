# CollabSpace

A real-time collaborative workspace platform built with React, TypeScript, Node.js, Express, MongoDB, and Socket.IO.

## Features

* Real-time collaboration
* Multi-user workspaces
* Authentication & Authorization
* Workspace management
* Comments & discussions
* Version tracking

## Tech Stack

### Frontend

* React
* TypeScript
* Tailwind CSS
* Socket.IO Client

### Backend

* Node.js
* Express.js
* Socket.IO
* JWT Authentication

### Database

* MongoDB
* Mongoose

## Installation

### Backend

```bash
npm install
npm run dev
```

### Frontend

```bash
npm install
npm run dev
```

## Environment Variables

```env
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
PORT=5000
```
